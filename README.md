### Десктопный чат

### Запуск проекта:

Для запуска проекта вам необходимо набрать команду в терминале:

 `npm i`

После набрать команду:

 `npm start`

### О проекте:

- Отправка сообщений
- Поиск сообщений
- Поиск контактов
- Выбор чата
- Профиль активного чата

### Технологии проекта
